
# 📊 Vrinda Sales Analysis

## 📌 Project Overview
This project analyzes sales data of Vrinda Store using Microsoft Excel.  
The goal is to identify customer behavior, sales trends, and business insights using pivot tables and an interactive dashboard.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  

---

## 📂 Dataset Features
The dataset includes:
- Order ID, Customer ID  
- Gender, Age, Age Group  
- Date, Month  
- Status, Channel  
- SKU, Category, Size  
- Quantity, Amount  
- Ship City, Ship Code  
- B2B  

---

## 📊 Key Analysis
- Sales by Order Channel  
- Age & Gender Analysis  
- State-wise Sales  
- Order Status  
- Men vs Women Comparison  

---

## 📈 Key Insights
- Adult customers contribute the highest number of orders  
- Women account for 52% of total orders across all age groups  
- Women contribute 61% of orders when compared only with men  
- Amazon is the top-performing sales channel  
- Maharashtra has the highest number of orders  
- 94% of orders are successfully delivered  

---

## 📊 Dashboard Features
- Interactive charts  
- Filters using slicers:
  - Month  
  - Channel  
  - Category
